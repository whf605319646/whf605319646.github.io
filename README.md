## 介绍
1. 博客初衷是打造一个好用的、面向程序员的`知识管理工具`
2. 轻松构建一个`结构化`的知识库，让你的知识海洋像一本本书一样清晰易读。
3. 博客功能提供一种知识的`碎片化`形态，并支持个性化博客配置。
4. `简洁高效`，以 Markdown 为中心的项目结构。内置自动化工具，以更少的配置完成更多的事。配合多维索引快速定位每个知识点。
