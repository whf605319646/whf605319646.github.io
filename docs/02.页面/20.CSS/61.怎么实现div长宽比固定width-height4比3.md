## 实现div宽长4比3

#### 方法一：
```html
<div class="cont"></div>
```
```css
.cont{
    width: 300px;
    background-color: #00a0e9;
}
.cont:after{
    display: block;
    width: 100%;
    padding-top: 75%;
    content: '';
}
```

#### 方法二：
```html
<div class="cont1"></div>
```
```css
.cont1{
    width: 40vw;
    height: 30vw;
    background-color: #ff5555;
}

```
